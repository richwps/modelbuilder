Contains dialogs for performing various wps::* and richwps::*-requests. Among these
  • ExecuteDialog - Execute any WPS process.
  • ExecuteModelDialog - Execute opened model as WPS process (if deployed).
  • ProfileModelDialog - Profile opened model as WPS process .
  • TestModelDialog - Test opened model as WPS process .
  • UndeployDialog - Undeploy any deployed model from an RichWPS server.
