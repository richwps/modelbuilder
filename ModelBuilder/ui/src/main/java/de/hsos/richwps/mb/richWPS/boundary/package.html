Interface to bordering RichWPS servers.

Used to wps-/wpst-Requestss against a given RichWPS server. Therefore it makes use of ExecuteRequestDTO and DeployRequestDTO.

As the connect-Methods imply, a RichWPS server is uniquely identified by a wps-endpoint. A RichWPS server wit transactional capabilities is instantiated with both, the wps- and wpst-endpoint.

ExecuteRequestDTO Object that can be used to perfom an wps:describeProcess()- and wps:execute()-Request. Object that starts out with an endpoint and a processidentifier. The RichWPSProvider can then be used to describe the remote process by retreiving according IInputSpecifiers and IOutputSpecifiers. Based on those descriptions IInputArguments and IOutputArguments can be set. Once set, the RichWPSProvider is able to perform an wps:execute()-Request and to obtain the resulting data.

DeployRequestDTO Object that can be used to perform an wpst:deploy()-Request.
